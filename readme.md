# 99Tech Code Challenge #1

This repository contains my solutions for the 99Tech Backend Engineer Code Challenge.
All required modules have been completed and are located in the `src` directory.

## Challenge Modules

### [Problem 4: Three Ways to Sum to N](./src/problem4/sum_to_n.ts)
Three distinct implementations of a function to sum integers from 1 to `n` in TypeScript, including a breakdown of their algorithmic complexities.

### [Problem 5: A Crude Server](./src/problem5/readme.md)
A fully functional REST API built with Express, TypeScript, and SQLite. Features include Layered Architecture, Zod strict backend validation, Swagger OpenAPI documentation, structured logging with Winston, and a complete Jest integration test suite.

### [Problem 6: Architecture](./src/problem6/readme.md)
An architectural specification and execution diagram for a real-time Scoreboard API Module, leveraging WebSockets and Redis for Top-10 broadcasting, with a strong focus on secure, synchronous authentication and execution validation.
